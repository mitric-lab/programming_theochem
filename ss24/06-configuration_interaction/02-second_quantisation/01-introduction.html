Programming in Theoretical Chemistry

Introduction

Quantum states of the first quantisation live in the Hilbert space, where the number of particles is conserved. Because we want to have a description that allows for the creation and annihilation of particles, we need to move to a larger vector space, the Fock space. In this section, we will introduce the Fock space and the operators that act on vectors in it.

The Fock Space

We already know that a first quantisation state with N particles lives in the Hilbert space A^H⊗N, where A^ is the antisymmetrisation operator and H is a single-particle Hilbert space. The notation H⊗N means that we take the tensor product of H with itself N times, i.e. H⊗N=n timesH⊗⋯⊗H​​

Since we want to allow an arbitrary number of particles in second quantisation, we must enter a larger space, the so-called Fock space. The Fock space is defined as the direct sum of all N-particle Hilbert spaces: F=N=0⨁∞​H⊗N We also include the N=0 case, which only contains the vacuum state ∣vac⟩. A determinant is represented by an occupation-number (ON) vector k, ∣k⟩=∣k1​,k2​,⋯,kM​⟩,kp​∈{0,1} with M single-particle states. The occupation numbers kp​ can either be 0 (empty) or 1 (occupied) due to the Pauli exclusion principle. The single-particle state is also called a site and can e.g. be an orbital.

Because we do not want to do difficult calculations with state vectors (except for the vacuum state), we must represent states by some (combination of) operators, something that generate all possible states in the Fock space from the vacuum state.

Creation and Annihilation Operators

We want to define a creation operator ap†​ that creates a particle at site p. It should also take care of Pauli exclusion, i.e. it should destroy the state if site p is already occupied. This leads to the following definition: ap†​∣k1​,⋯,0p​,⋯,kM​⟩ap†​∣k1​,⋯,1p​,⋯,kM​⟩​=Γpk​∣k1​,⋯,1p​,⋯,kM​⟩=0​​ where the phase factor Γpk​ is defined as Γpk​=q=1∏p−1​(−1)kq​ So for every already occupied site q before site p, we have to multiply the state with −1. This is necessary to ensure antisymmetry.

These two equations can be combined into a single one: ap†​∣k⟩=δkp​,0​Γpk​∣k1​,⋯,1p​,⋯,kM​⟩ From the definition, it trivially follows that ap†​ap†​∣k⟩=0

For two different creation operators ap†​ and aq†​, we can show the following: (ap†​aq†​+aq†​ap†​)∣k⟩=0

Proof

Without loss of generality, we can assume p<q. It follows that ap†​aq†​∣⋯,kp​,⋯,kq​,⋯⟩​=ap†​δkq​0​Γqk​∣⋯,kp​,⋯,1q​,⋯⟩=δkp​0​δkq​0​Γpk​Γqk​∣⋯,1p​,⋯,1q​,⋯⟩​​ since the phase factor Γpk​ is not affected by the application of aq†​. Reversing the order of both operators, we obtain aq†​ap†​⋯,kp​,⋯,kq​,⋯⟩​=aq†​δkp​0​Γpk​∣⋯,1p​,⋯,kq​,⋯⟩=δkp​0​δkq​0​Γpk​(−Γqk​)∣⋯,1p​,⋯,1q​,⋯⟩​​

This time, an extra factor of −1 appears since the application of ap†​ changes the parity for aq†​. Combining the two equations above, we can write (ap†​aq†​+aq†​ap†​)∣k⟩=0 which also holds for the case p=q.

The action of Hermitian adjoints ap​ on ∣k⟩ can be understood by inserting the identity ap​∣k⟩=m∑​∣m⟩⟨m∣ap​∣k⟩.

The matrix element that has occured can be evaluated as ⟨m∣ap​∣k⟩=⟨k∣ap†​∣m⟩∗=δmp​0​Γpm​⟨k∣m1​,⋯,1p​,⋯mM​⟩∗.

The overlap ⟨k∣m1​,⋯,1p​,⋯⟩ is nonzero if and only if both vectors are identical, which is the case when ]k⟩ and ∣m⟩ only differ at position p, where kp​=1 and mp​=0. Therefore, this matrix element can also be written as

⟨m∣ap​∣k⟩​=δmp​0​δkp​1​Γpm​⟨k1​,⋯,0p​,⋯kM​∣m1​,⋯,1p​,⋯mM​⟩∗=δmp​0​δkp​1​Γpm​⟨m1​,⋯,1p​,⋯mM​∣k1​,⋯,1p​,⋯kM​⟩=δmp​0​δkp​1​Γpm​⟨m1​,⋯,0p​,⋯mM​∣k1​,⋯,0p​,⋯kM​⟩=δkp​1​Γpm​⟨m∣k1​,⋯,0p​,⋯kM​⟩=δkp​1​Γpk​⟨m∣k1​,⋯,0p​,⋯kM​⟩,​​ where we used the fact Γpk​=Γpm​, which follows directly from its definition. Therefore, only one term in the sum survives and we can conclude ap​∣k⟩=δkp​1​Γpk​∣k1​,⋯,0p​,⋯kM​⟩.

The operator ap​ removes the occupation at p if occupied and returns zero if unoccupied. Therefore, we shall call it annihilation operator. A special case of the equation above is ap​∣vac⟩=0, which states that the annihilation operator destroys the vacuum state.

Since the annihilation operator is the Hermitian adjoint of the creation operator, we can follow (ap​aq​+aq​ap​)∣k⟩=0.

We can then examine the action of the action of one creation and one annihilation operator on a state vector ∣k⟩. After several algebraic transformations, it can be shown that (ap†​aq​+aq​ap†​)∣k⟩=δpq​.

Proof

For p=q, we have ap†​ap​∣⋯,kp​,⋯⟩​=ap†​δkp​1​Γpk​∣⋯,0p​,⋯⟩=δkp​1​(Γpk​)2∣⋯,1p​,⋯⟩=δkp​1​∣⋯,1p​,⋯⟩=δkp​1​∣k⟩​​ and ap​ap†​∣⋯,kp​,⋯⟩​=ap​δkp​0​Γpk​∣⋯,1p​,⋯⟩=δkp​0​Γpk​∣⋯,0p​,⋯⟩=δkp​0​∣⋯,0p​,⋯⟩=δkp​0​∣k⟩.​​ Adding both equations, we obtain (ap†​ap​+ap​ap†​)∣k⟩=(δkp​1​+δkp​0​)∣k⟩=∣k⟩, or, because the state vector ∣k⟩ is arbitrary, we can write (ap†​ap​+ap​ap†​)=1.

For p<q, we have ap†​aq​∣⋯,kp​,⋯,kq​,⋯⟩​=ap†​δkq​1​Γqk​∣⋯,kp​,⋯,0q​,⋯⟩=δkp​0​δkq​1​Γpk​Γqk​∣⋯,1p​,⋯,0q​,⋯⟩​​ and aq​ap†​∣⋯,kp​,⋯,kq​,⋯⟩​=aq​δkp​0​Γpk​∣⋯,1p​,⋯,kq​,⋯⟩=−δkp​1​δkq​0​Γpk​Γqk​∣⋯,1p​,⋯,0q​,⋯⟩.​​ The minus sign arises because the application of aq​ is affected by the previous application of ap†​, which changes the occupation of site p that comes before site q. Adding both equations, we obtain (ap†​aq​+aq​ap†​)∣k⟩=0, or (ap†​aq​+aq​ap†​)=0.

The case p>q can be obtained by complex conjugation of the case p<q and exchanging the dummy indices p and q.

Combining all three cases, we obtain (ap†​aq​+aq​ap†​)=δpq​.

We have now worked out the action of two ladder operators (creation or annihilation operator) on an arbitrary state vector. Since it works for all possible states, we can as well write the equations solely in terms of operators: {ap†​,aq†​}{ap​,aq​}{ap†​,aq​}​=0=0=δpq​​​ where {A,B}=AB+BA is the anticommutator. These relations are known as the anticommutation relations of the ladder operators and can be viewed as the defining property of the ladder operators.

Now we know how to represent states using operators in second quantisation. But how do observables look like?

Operators in Second Quantisation

We shall at first take a look at one-particle operators O^1​=∑i​o^(i), where o^(i) is the one-particle operator acting on the i-th particle. By inserting the identity to the left and right of the operator, we obtain O^1​=i∑​o^(i)=i∑​k,k′∑​∣k⟩⟨k∣o^(i)∣k′⟩⟨k′∣

Suppose the i-th particle is on the p-th site in the state ∣k⟩ and q-th site in the state ∣k′⟩. Then we can replace the sum over i by a sum over p and q. Because the operator o^(i) acts only on the i-th particle, the matrix element ⟨k∣o^(i)∣k′⟩ evaluates to ⟨k∣o^(i)∣k′⟩=⟨k∖p∣⟨p∣o^(i)∣q⟩∣k′∖q⟩=opq​⟨k∖p∣k′∖q⟩ where ∣k∖p⟩ (sloppily) denotes the state vector ∣k⟩ with the p-th site removed. But we already know how to (formally) remove a site from a state vector, namely by applying the annihilation operator ap​. Therefore, we can write the overlap as ⟨k∖p∣k′∖q⟩=⟨ap​k∣aq​k′⟩=⟨k∣ap†​aq​∣k′⟩

Putting everything together, we obtain O^1​=pq∑​opq​ap†​aq​

We can see that a one-particle operator is represented by linear combination of creation-annihilation pairs weighted by the matrix elements of the one-site operator o^(i).
